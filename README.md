🌻 Mini Plant vs Zombies (Python Pygame)
Một bản clone rút gọn của game Plant vs Zombies huyền thoại, được viết hoàn toàn bằng Python và thư viện Pygame. Game sử dụng thuật toán vẽ hình học cơ bản, không cần tải thêm hình ảnh hay âm thanh (No assets required).

✨ Tính năng (Features)
4 loại cây: Peashooter (Bắn đậu), Sunflower (Sinh mặt trời), Wallnut (Thịt chắn), Cherry Bomb (Nổ diện rộng).
3 loại Zombie: Zombie thường, Zombie cầm Nón (nhiều máu), Zombie xô sắt (cực nhiều máu).
Hệ thống Wave (Sóng quái): 5 đợt quái, đợt cuối là "Huge Wave" (Siêu sóng).
Máy cắt cỏ (Lawnmowers): Tự động kích hoạt khi zombie đi hết sân.
Xẻng (Shovel): Dùng để bứng cây đã trồng.
Hệ thống Cooldown: Thời gian hồi chiêu cho từng loại cây.
🎮 Cách chơi (How to play)
Thu thập "Mặt trời" (Sun) rơi từ trên trời hoặc từ cây Sunflower để có tiền.
Click chọn cây ở thanh menu trên cùng.
Click vào ô cỏ trống để trồng cây.
Tiêu diệt toàn bộ zombie trước khi chúng đi hết sân cỏ bên trái.
💻 Cài đặt và Chạy (Installation)
Đảm bảo bạn đã cài Python 3.x.
Cài đặt thư viện Pygame:
pip install -r requirements.txt
(Hoặc chạy: pip install pygame)
3. Chạy game:

bash

python pvz_full.py
